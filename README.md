# Niuvis（牛维斯）

操作系统级 AI 助手，参考腾讯 Marvis 的产品理念，打造一款能够深度操控本地系统、文件和应用的个人 AI Agent。

## 项目定位

Niuvis 不是一个聊天机器人，而是一个**操作系统层级的 AI 中间层**。用户通过自然语言下达指令，Niuvis 理解任务、拆解步骤、调用对应 Agent 执行，在涉及隐私、安全等关键环节时交回用户确认。

## 核心能力

### 系统级操控
- 查询电脑配置、电池健康、网络状态
- 修改系统设置（分辨率、电源计划、开机启动项等）
- 清理冗余文件、优化系统性能
- 硬件检测与兼容性分析

### 文件智能管理
- 语义搜索：基于文件内容而非文件名查找
- 自动归档：按时间、类型、内容智能分类
- 批量处理：压缩、重命名、格式转换
- 文档理解：总结、提取关键信息、生成报告

### 应用操控
- 调用和操作本地 EXE 应用
- 自动化操作流程（如定时签到、监控等）

### 跨端协同
- 支持 Windows / macOS / iOS / Android 多端入口
- 手机可远程查看和控制电脑
- 同账号任务互通

## Agent 架构

```
┌─────────────────────────────────────────────┐
│                 用户指令                      │
└──────────────────────┬──────────────────────┘
                       │
                       ▼
┌──────────────────────────────────────────────┐
│              主管 Agent（Orchestrator）        │
│  理解需求 → 拆解任务 → 分配 Agent → 汇总结果  │
└───┬──────┬──────┬──────┬──────┬──────────────┘
    │      │      │      │      │
    ▼      ▼      ▼      ▼      ▼
┌─────┐┌─────┐┌─────┐┌─────┐┌─────┐
│File ││Sys  ││App  ││Web  ││Search│
│Agent││Agent││Agent││Agent││Agent │
└─────┘└─────┘└─────┘└─────┘└─────┘
```

| Agent | 职责 |
|-------|------|
| 主管 Agent | 理解意图、分配任务、协调执行、汇总结果 |
| File Agent | 文件搜索、读写、归档、格式转换、文档分析 |
| Sys Agent | 系统设置、硬件检测、配置修改、性能优化 |
| App Agent | 调用本地应用、自动化操作、进程管理 |
| Web Agent | 网页交互、数据抓取、表单填写 |
| Search Agent | 网络搜索、信息聚合、实时查询 |

## 运行模式

### 效率模式（端云协同）
- 复杂推理调用云端大模型（DeepSeek / GPT 等）
- 文件处理、本地索引在端侧完成
- 响应快、能力强，适合日常办公

### 隐私模式（纯端侧）
- 使用端侧模型（如 Qwen、Llama 等）
- 所有数据本地处理，完全不上云
- 断网可用，适合财务、法务等敏感场景

## 技术架构

```
┌───────────────────────────────────────────────┐
│              Electron 客户端                   │
│  ┌─────────────────────────────────────────┐  │
│  │          渲染进程（React + TS）          │  │
│  │    UI 界面 / 任务面板 / 对话窗口         │  │
│  └────────────────────┬────────────────────┘  │
│                       │ IPC                    │
│  ┌────────────────────▼────────────────────┐  │
│  │          主进程（Node.js + TS）          │  │
│  │  Agent 调度 / 系统操作 / 文件管理        │  │
│  │  模型 API 直连（用户配置 API Key）       │  │
│  └────────────────────┬────────────────────┘  │
└───────────────────────┼────────────────────────┘
                        │
                        ▼
┌───────────────────────────────────────────────┐
│             模型提供商 API                     │
│  OpenAI / DeepSeek / 混元 / Claude / Ollama   │
└───────────────────────────────────────────────┘
```

> 开源项目，不提供后端服务。用户自行配置 API Key，客户端直连模型提供商。

## 配置说明

用户需在设置页面配置以下信息：

| 配置项 | 说明 |
|--------|------|
| 模型提供商 | OpenAI / DeepSeek / 混元 / Claude / Ollama（本地） |
| API Key | 对应提供商的 API 密钥 |
| 模型名称 | 如 gpt-4o、deepseek-chat、qwen-plus 等 |
| API 地址 | 可选，用于自定义代理或私有部署 |

支持同时配置多个模型，按场景自动切换：
- 效率模式：使用云端大模型（GPT-4o / DeepSeek V4）
- 隐私模式：使用本地模型（Ollama）
